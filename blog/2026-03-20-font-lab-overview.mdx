# Font Lab Overview

Trang nay dung de test nhanh bo giao dien moi voi:

- Bai viet dai, nhieu heading de check TOC va hierarchy.
- Font options khac nhau cho doc tieng Viet + tieng Anh.
- Cach doc tren desktop/mobile khi bai viet co nhieu section.

## Muc tieu test

1. Kiem tra do de doc khi bai dai tren trang blog.
2. So sanh cam giac typography giua 3 style.
3. Danh gia heading spacing, paragraph rhythm, va code block readability.

## Cach test de nhanh co ket qua

### Step 1: Desktop readability

- Doc lien tuc 2-3 section trong moi bai con.
- Check xem mat co met khi doc paragraph dai hay khong.
- Check heading co du noi bat de scan noi dung khong.

### Step 2: Mobile readability

- Mo trang tren man hinh hep.
- Check do dai dong, khoang cach heading, va tam vung bam link.
- Danh gia TOC co de follow hay khong.

### Step 3: Bilingual content

- Vua doc doan tieng Viet, vua doc doan tieng Anh o cung section.
- Check font co giu duoc dau tieng Viet ro va dep khong.
- Check nhung tu ky thuat tieng Anh co bi vo nhip line-height khong.

## Checklist review

- [ ] Heading H2/H3/H4 co de scan.
- [ ] Paragraph dai van de doc va khong bi sat dong.
- [ ] Link, blockquote, code block de nhan ra.
- [ ] Font van dep khi co dau tieng Viet day du.
- [ ] Trang mobile khong bi chan chu hoac qua chat.

## Ghi chu

Neu ban thay option nao on nhat, minh co the dat option do lam default va them theme toggle de doi font ngay tren giao dien.
