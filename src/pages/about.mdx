# About Tech Briefing

Tech Briefing la noi doc nhanh xu huong cong nghe moi nhat, theo format gon, ro, va co link bai goc de dao sau khi can.

> Neu ban muon cap nhat tin cong nghe nhanh trong 3-5 phut moi ngay, day la feed danh cho ban.

## Ban nhan duoc gi?

- Tong hop tin tu cac nguon cong nghe uy tin (The Verge, Ars Technica, TechCrunch, ...).
- Tom tat ngan gon theo cau truc "TL;DR -> Why it matters -> Source".
- Link den bai goc de kiem chung va mo rong context.
- Luong noi dung deu theo lich ingest tu dong.

## Quy trinh xuat ban

1. Lay bai moi tu danh sach RSS da cau hinh.
2. Loc trung + chuan hoa metadata.
3. Tao tom tat ngan de doc nhanh.
4. Sinh Markdown va publish qua Docusaurus.
5. Build va deploy len GitHub Pages.

## Nguyen tac

- Ton trong robots.txt va Terms of Use cua tung nguon.
- Luon dan link ve bai goc.
- Khong copy lai nguyen van noi dung dai.
- Uu tien tong hop va goc nhin trung lap.

## Tan suat cap nhat

- Feed ingest chay dinh ky moi 6 gio.
- Bai moi duoc dua len trang chu ngay sau khi pipeline cap nhat.

## Van hanh

Cau hinh nguon tai `config/sources.json`.

Chay ingest local:

```bash
cd hutune-blog
npm run ingest
```

Build site:

```bash
npm run build
```
