# About Tech Briefing

Tech Briefing la blog tong hop tin cong nghe theo dinh ky.

Noi dung duoc xu ly theo pipeline:

1. Lay bai moi tu danh sach nguon RSS da cau hinh.
2. Loc trung va chuan hoa metadata.
3. Tao tom tat ngan gon.
4. Sinh file Markdown de publish qua Docusaurus.

Muc tieu la giu trang gon, doc nhanh, va co link den bai goc de doi chieu.

## Nguyen tac

- Ton trong robots.txt va Terms of Use cua tung nguon.
- Luon dan link ve bai goc.
- Khong copy lai nguyen van noi dung dai.
- Uu tien tong hop va goc nhin trung lap.

## Van hanh

Cau hinh nguon tai `config/sources.json`.

Chay ingest local:

```bash
cd hutune-blog
npm run ingest
```

Build site:

```bash
npm run build
```
